# DataStructures408Cpp

C++ practice project for learning data structures and preparing for the 408 exam.

## Directory Layout

- `linear_list/`: sequential list, linked list, double linked list, circular linked list
- `stack_queue/`: stack, queue, circular queue, deque
- `string/`: string matching, KMP
- `array_generalized_list/`: array, matrix, sparse matrix, generalized list
- `tree_binary_tree/`: tree, binary tree, binary search tree, heap, Huffman tree
- `graph/`: graph storage, traversal, shortest path, minimum spanning tree, topological sort
- `search/`: sequential search, binary search, block search, hash table
- `sort/`: insertion sort, exchange sort, selection sort, merge sort, radix sort
- `notes/`: chapter notes and 408 key points
- `tests/`: focused test programs and examples

## Build

```bash
cmake -S . -B cmake-build-debug
cmake --build cmake-build-debug
```
